# 🚗 Car Rental App

A fully **responsive car rental application** built with modern web technologies. Users can browse available rental cars, search for specific vehicles, and view detailed information for each car.

## 🚀 Technologies Used

* React
* TypeScript
* react-router-dom
* react-select
* react-paginate
* motion
* Tailwind CSS
* OpenDataSoft API

## 📌 Features

* 🚗 **Car Listings:** Browse rental cars displayed in a clean card layout.
* 🔍 **Search Functionality:** Easily find cars based on user input.
* 📄 **Car Detail Page:**

  * Price
  * Brand
  * Model
  * Model Year
  * Car images
  * Additional vehicle details
* 🖱️ **Interactive UI:** Click on any car card to view detailed information.
* 📑 **Pagination:** Navigate through multiple pages of car listings.
* 🎨 **Modern UI:** Styled with Tailwind CSS for a clean and user-friendly design.
* 📱 **Responsive Design:** Works seamlessly across mobile, tablet, and desktop devices.
* ⚡ **API Integration:** Data fetched dynamically using OpenDataSoft API.
